Frequently Asked Questions

What does a '?' symbol mean in Date and Competition column?

A question mark is displayed if there is no information about the next meet-up.

How can I add a match?

At the moment, there is no method for visitors to add nor request matches. It is possible that this functionality will be added in the future.

I can't see any matches! What's wrong?

To display a table, the website uses DataTables plugin. Please, make sure your adblocking extensions are not blocking the website from accessing it.
